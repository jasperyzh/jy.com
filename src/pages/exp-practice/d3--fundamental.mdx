# Introduction to D3.js Fundamentals and SVG for Web Visualization

This workshop provides a solid foundation in **Scalar Vector Graphics (SVG)** for drawing custom shapes and understanding its syntax. The primary focus is on the fundamental parts of **D3.js (Data-Driven Documents)**, the de facto JavaScript library for web data visualization, teaching core principles like selections, data binding, scales, and layouts.

### SVG Primitives and Coordinate System

- **SVG Paths:** Used to draw custom shapes (like a flower petal) via the `d` (define) attribute, requiring an understanding of the SVG coordinate system.
- **SVG Transforms:** Used for positioning and orientation, specifically the `translate` command (for positioning each petal) and the `rotate` command (for arranging petals into a flower).

### D3 Selections and Data Binding

- **D3 Selections and Binding:** The initial step involves using D3 to select existing elements (like a petal) and bind movie data to them.
- **The Enter-Append Pattern:** Creating an element for each data point is achieved using D3's `enter-append` pattern.
- **Nesting Selections:** Elements can be grouped for complex arrangements (e.g., turning individual petals into complete flowers) using D3 selection nesting.

### Data-to-Visual Mapping with Scales

- **D3 Scales:** Essential for translating raw data (e.g., IMDb rating, number of votes) into visual properties (e.g., size, number of petals) that can be drawn on the screen. Functions like `d3.min`, `d3.max`, and `d3.extent` help analyze the data domain.

### Interactivity and Performance

- **The Update-Exit Pattern and Transitions:** When data changes (like a real-time feed), the `enter-update-exit` pattern handles updating the DOM elements, often paired with D3 transitions to animate smoothly from the old data state to the new state.
- **D3 Layouts:** D3 provides various methods to lay out data, including shapes and hierarchies, and the `d3-force` layout is used for positioning elements (like the film flowers) by genre.

## Key Insights

- **Connections:** D3 connects JavaScript data manipulation with the DOM via SVG, serving as the bridge for data-driven visualization. The ecosystem has shifted from static code blocks to dynamic notebooks.
- **Actionable Insights:** The D3 API is comprehensive and can be overwhelming, so focusing on core principles allows developers to comfortably read and understand the multitude of community examples.
