Interactive Web Component Demo

This is a sample sketch demonstrating the sketches feature. In a real sketch, you would include more detailed content and possibly embed interactive components.

<!-- Example HTML component -->
<div class="interactive-component">
  <h2>Interactive Component</h2>
  <button id="demo-button">Click Me!</button>
  <div id="output"></div>
</div>

/* Example CSS */
.interactive-component {
  padding: 20px;
  border: 1px solid #ccc;
  border-radius: 4px;
  margin-top: 20px;
}

#demo-button {
  background: #0070f3;
  color: white;
  border: none;
  padding: 8px 16px;
  border-radius: 4px;
  cursor: pointer;
}

#demo-button:hover {
  background: #005cdd;
}

// Example JavaScript
document.addEventListener('DOMContentLoaded', () => {
  const button = document.getElementById('demo-button');
  const output = document.getElementById('output');
  
  let clickCount = 0;
  
  button.addEventListener('click', () => {
    clickCount++;
    output.textContent = `Button clicked ${clickCount} time${clickCount === 1 ? '' : 's'}!`;
  });
});

How This Works

In a real sketch, you might include explanations of your code, implementation details, or design decisions. You can also embed interactive components from your components directory.

For example, if you had a component at src/components/sketches/DemoComponent.vue, you could import and use it like this:

import DemoComponent from '@components/sketches/DemoComponent.vue';
